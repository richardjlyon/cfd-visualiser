# Phase 1 — UI Design Contract

> Visual and interaction contract for the first public-facing page of the CfD Visualiser: an Observable Framework site hosting CHART-01 (the "scissors" strike-vs-market time series) with editorial prose, source citation, last-updated stamp, and downloadable JSON. Generated by gsd-ui-researcher; verified by gsd-ui-checker.

> **shadcn gate:** NOT APPLICABLE. Stack is Observable Framework + Pico CSS (Markdown-first static site). React/Next.js/Vite are explicitly rejected by CLAUDE.md "What NOT to Use". No `components.json`, no component registry.

## Spacing Scale

8-point scale (multiples of 4). Pico CSS's default rhythm already uses rem-based spacing compatible with this grid; these tokens pin the exact values for chart-specific CSS.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Inline gaps (e.g. tooltip padding), icon-text gaps |
| sm | 8px | Compact element spacing (caption-to-source gap) |
| md | 16px | Default element spacing; chart-to-caption gap; paragraph rhythm (Pico default `1rem`) |
| lg | 24px | Section padding inside boxouts; chart-to-boxout gap |
| xl | 32px | Chart-to-next-section gap; header-to-main gap |
| 2xl | 48px | Major section breaks (between chart stack and footer) |
| 3xl | 64px | Page-level top/bottom padding on desktop |

**Exceptions:**
- Minimum tap target: 44×44px (WCAG 2.5.5 AAA, required by OPS-03). Applies to the round-toggle controls and the download-JSON link. 44 is not on the 8-point grid; it is an accessibility override.
- Content column max-width: **720px** on screens ≥768px (readable measure for prose + gives charts a natural constrained width). Pico's default container caps at ~1130px — we narrow it for this editorial layout.

## Typography

Four sizes, two weights. All sizes expressed in px but implemented as rem (Pico default root = 16px; these values resolve correctly).

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body / caption / source line | 16px | 400 (regular) | 1.5 |
| Chart axis labels / tooltip / boxout label | 14px | 400 (regular) | 1.4 |
| Chart title / h2 | 20px | 600 (semibold) | 1.3 |
| Page title / h1 | 28px | 600 (semibold) | 1.2 |

**Rules:**
- No more than 4 sizes, no more than 2 weights site-wide in Phase 1.
- Pico default `small` (≈0.875rem = 14px) is reused for the source-citation line — no override needed.
- Chart tick labels use 14px; chart axis titles use 14px semibold. Plot's default 10px tick size is **overridden** via `fontSize: 14`.
- Italics reserved for the source-citation line only (matches editorial convention).

## Color

Dominant / secondary / accent split. Palette is **Okabe-Ito colourblind-safe** (required by OPS-03). Values below are light-mode; Pico's auto dark mode swaps surface colours to dark equivalents and the data palette stays unchanged (all three data hues remain AA-contrast on both white and Pico's dark surface `#11191f`).

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#ffffff` (light) / `#11191f` (dark) — Pico defaults | Page background, chart plot area |
| Secondary (30%) | `#f1f3f5` (light) / `#1a232b` (dark) — Pico `--pico-muted-bg` | "What does this mean?" boxout background; footer; any card surface |
| Accent (10%) | `#0072B2` (Okabe-Ito blue) | **Strike price line only**, plus links, plus the "Download JSON" link |
| Data: market reference price | `#E69F00` (Okabe-Ito orange) | Market reference price line |
| Data: subsidy gap fill | `#D55E00` (Okabe-Ito vermillion) at 18% opacity | Area fill between strike and market lines when strike > market (positive subsidy) |
| Data: clawback fill | `#009E73` (Okabe-Ito green) at 18% opacity | Area fill between strike and market when market > strike (negative subsidy / 2022 clawback) |
| Neutral gridline | `#dddddd` (light) / `#2a3842` (dark) | Chart gridlines, axis ticks |
| Text primary | Pico default (`#212121` light / `#e1e1e1` dark) | Body copy |
| Text muted | Pico `--pico-muted-color` | Source line, last-updated stamp, axis tick labels |
| Destructive | not used in Phase 1 | No destructive actions exist in Phase 1 (view-only page) |

**Accent reserved for:**
1. The strike-price line in CHART-01 (the "what consumers are locked into" signal)
2. Hyperlinks in prose (Pico default behaviour)
3. The "Download JSON" link
4. Focus ring on interactive controls (keyboard accessibility, Pico default)

**Explicitly NOT accent-coloured:** round-toggle buttons (default state uses neutral border; active state uses accent fill), caption text, source line, boxout heading. Accent must remain rare to retain signal value.

**Colour-only encoding is FORBIDDEN** (OPS-03). The subsidy gap is communicated by:
1. The two-line geometry (always visible)
2. The fill colour (red vs green)
3. Direct tooltip readout: "Subsidy: £X/MWh" with explicit sign

## Copywriting Contract

This page has no transactional CTA (it is an editorial read-only chart page). The primary call-to-action is the round-toggle filter and the Download JSON link.

| Element | Copy |
|---------|------|
| Page title (h1) | `UK CfD: what consumers pay vs the market` |
| Chart title (h2) | `Strike price vs market reference price, {start_year}–{end_year}` (dates interpolated at build time from view-model) |
| Caption (EDIT-01) | `CfD strike prices have tracked above wholesale prices in every year except 2022 — consumers paid the gap.` (factual voice; no loaded words per EDIT-05) |
| Source line (EDIT-02) | `Source: LCCC Actual CfD Generation and Avoided GHG Emissions dataset. Last updated {ISO date}.` with the dataset name linked to `https://dp.lowcarboncontracts.uk/dataset/actual-cfd-generation-and-avoided-ghg-emissions`. |
| "What does this mean?" boxout heading (EDIT-04) | `What does this mean?` |
| Boxout body (EDIT-04) | `The strike price is what generators are guaranteed per unit of electricity. The market reference price is the going wholesale rate. Consumers pay the difference through their bills — that gap is the subsidy.` (plain language, editorial framing allowed here per EDIT-05) |
| Round-toggle legend label | `Show allocation rounds:` |
| Round-toggle options | `All`, `Investment Contracts`, `AR1`, `AR2`, `AR4`, `AR5` (no "AR3" — cancelled; see RESEARCH schema notes). `All` is the default. |
| Download link | `Download this chart's data (JSON)` — links to `/data/chart-3c.json` |
| Tooltip on chart hover | Three lines: `{date}` / `Strike: £{strike}/MWh` / `Market: £{market}/MWh` / `Subsidy: £{gap}/MWh` (gap shown with sign — negative during clawback) |
| Empty state heading | `Chart data unavailable` |
| Empty state body | `The daily pipeline did not produce a view-model for this chart. Check the last-updated stamp above; if it is stale, the latest build failed. The previous snapshot remains visible.` (empty state is rare — static build means either the chart renders or the prior deploy is still live) |
| Error state | `Something went wrong loading this chart.` with console-level detail only; no raw stack traces in the UI. Retry by reloading. |
| Destructive confirmation | not applicable (no destructive actions in Phase 1) |
| Glossary entries (one-line gloss, per CLAUDE.md "no jargon" rule) — stored in `src/content/glossary.json` for reuse | `CfD`: "Contract for Difference — a government scheme that guarantees renewable generators a fixed price per MWh." / `Strike price`: "The guaranteed £/MWh a generator is paid under its CfD." / `Reference price`: "The market price the CfD is measured against." / `IMRP`: "Intermittent Market Reference Price — the wholesale benchmark used for wind and solar." / `Allocation round`: "A government auction round in which CfDs were awarded." |

**Editorial grammar rule (EDIT-05) enforced:** Caption, chart title, axis labels, and tooltip are **factual only**. Framing words (`waste`, `scandal`, `broken`, `unfair`) appear only in the boxout or surrounding prose, never inside the chart figure or its caption.

## Interactions

| Interaction | Behaviour |
|-------------|-----------|
| Round toggle | Client-side Observable Plot re-render driven by a Framework reactive cell; toggling a round filters the view-model to that allocation round's aggregate. `All` is default and shows the wind+solar IMRP aggregate. **Ships in Phase 1** (required by ROADMAP success criterion #1). |
| Hover tooltip | Plot `tip: true` on the strike and market line marks. Tooltip content per Copywriting Contract. Touch-tap equivalent on mobile (Plot handles natively). |
| Zoom | Pinch-zoom and drag-scroll via Plot's built-in `interval` brush on the x-axis. Keyboard-accessible via Plot's default focus handling. **Success criterion #1 requires this.** |
| Download JSON | Plain `<a href="/data/chart-3c.json" download>` — no JS. File is always the current view-model for the currently-selected round filter? **Phase 1 decision: download always serves the unfiltered all-rounds view-model** (simpler, more useful for analysts who want to run their own aggregations). Round-filter is a UI affordance only. |
| Focus ring | Pico default (accent colour, 2px ring, 2px offset) on all buttons and links. Never suppressed. |

## Accessibility

| Requirement | How Met |
|-------------|---------|
| WCAG AA contrast | Pico defaults are AA in both themes; data palette verified Okabe-Ito contrast on both surfaces |
| No colour-only encoding | Subsidy gap communicated via geometry + colour + tooltip readout |
| Keyboard navigation | All interactive controls reachable via Tab; focus ring always visible; round-toggle buttons are `<button>` elements |
| Chart ARIA | Plot figure wrapped in `<figure role="img" aria-labelledby="chart-3c-title" aria-describedby="chart-3c-caption">`; title + caption elements referenced |
| Min tap target | 44×44px on round-toggle and download link |
| Reduced motion | No animated transitions on initial render; Plot's default behaviour respected |
| Text alternative | Caption + source line + boxout together provide non-visual readers the core takeaway without needing to parse the SVG |
